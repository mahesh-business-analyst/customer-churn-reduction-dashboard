# Customer Churn Reduction Dashboard

## Overview

Business Analyst project analyzing customer churn using Power BI and SQL.

## Tools Used

* Power BI
* DAX
* SQL
* Excel

## Key Features

* KPI Cards
* Churn by Region & Plan Type
* Join Trends
* Insights & Recommendations

## SQL Analysis

Includes SQL queries for churn analysis and risk detection.
